A Simple Discord Script To Delete Your Messages In Servers And DMs.

## Guide

Put your Discord token inside the `.env` file.
   
### Example:
```
TOKEN=your_token_here
```

### Install The Required Dependencies:
```
pip install aiohttp python-dotenv
```

## Options

* ### DM Clear
Deletes every one of your messages in direct messages section, will clear only if dm is open (can close dm for it to skip).

* ### Channel Clear (Listing Method)
Deletes every message of yours in a selected channel, preferably in a small channel.

* ### Channel Clear (Search Method)
Deleted every message of youres in a selected channel, using the search method, can be used in any channel.

* ### Guild Clear
Deletes every one of your messages in a server of your choice.

## Notes

* This script deletes **only your own messages**.
* Works in **servers, channels and DMs**.
* Make sure your token stays **private** and never share it.
